# NanoBot AI Assistant - Home Assistant Add-on

[![NanoBot](https://img.shields.io/badge/NanoBot-v0.1.4-blue)](https://github.com/HKUDS/nanobot)

Run [NanoBot](https://github.com/HKUDS/nanobot) as a Home Assistant add-on.

## Features

- Ultra-lightweight AI assistant
- Multiple LLM provider support (OpenAI, Anthropic, custom endpoints)
- Discord bot integration
- Gateway API on port 18790
- Memory and context management
- Web search capabilities
